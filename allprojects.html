TEDxBrandeisUniversity

TEDxBrandeisUniversity is an independently organized and licensed TEDx event hosted at Brandeis University. The mission of our event is to inspire the community with thought-provoking TED talks that spark conversations and connections in our local and global communities.

TALK

TALK is an app-based platform that employs multilingual youth to provide live video-remote interpretation in businesses, schools, and nonprofits. This project was created initially as a project for Brandeis SPARK and ultimately as a regional finalist project for the Hult Prize.

GI Bill® School Feedback Tool

The GI Bill® School Feedback Tool is a platform that helps Veterans submit feedback about the school where they used their GI Bill benefits. This project was part of my fellowship at the Digital Services Team at Veteran Affairs.

DocFinder

DocFinder is a voice-enabled search application that provides a list of doctors specified by your search queries. Search can be done by your insurance provider, the type of doctor you are looking for, the location you like to search from, and languages.

TMBR

TMBR is an Android mobile application designed to promotes user safety in a contemporary setting, and was designed to be non-intrusive and run in the background of the phone.

CS Squared (CS^2)

CS Squared is a concept for a web application designed for the underrepresented minority students in Brandeis University's Computer Science department.
